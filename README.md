# TT Room 3D Reconstruction (Blender + Three.js)

This project is a 3D recreation of the TT Room, modeled in **Blender** and rendered using **Three.js** in a web viewer.  
It includes accurate room geometry, furniture modeling, lighting setup, and a clean, organized folder structure.
